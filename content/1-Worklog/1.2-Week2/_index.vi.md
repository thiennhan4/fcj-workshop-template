### Kết quả đạt được tuần 2:

* Hiểu các khái niệm cốt lõi về Điện toán đám mây:
  * Định nghĩa và đặc điểm của Cloud Computing
  * Các mô hình triển khai: Public Cloud, Private Cloud, Hybrid Cloud
  * Lợi ích chính: Khả năng mở rộng, Tính đàn hồi, Trả phí theo mức sử dụng

* Hiểu điều gì tạo nên sự khác biệt của AWS và nắm được Hạ tầng toàn cầu AWS:
  * Regions, Availability Zones, Edge Locations
  * AWS Management Console, CLI và SDK

* Tạo và bảo mật tài khoản AWS Free Tier thành công:
  * Bật MFA cho Root User bằng thiết bị MFA ảo
  * Tạo IAM Admin Group và IAM User theo nguyên tắc least privilege
  * Cấu hình xác thực tài khoản và chính sách mật khẩu

* Hoàn thành bài lab quản lý chi phí:
  * Tạo AWS Budget bằng template
  * Thiết lập cảnh báo cost và usage budget
  * Hiểu các mô hình định giá AWS và các gói AWS Support
